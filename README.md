# PyBind11Test


## Building
#### Toolchain
- GNU 7.1.0
- Cmake 3.9.2
- Python 3.6.2
- C++ 17
- PyBind11 2.2.0

#### Building with CMake
- Open a command prompt and cd to this projects root directory
- mkdir build
- cd build
- cmake .. -G"MinGW Makefiles"
- cmake --build .
